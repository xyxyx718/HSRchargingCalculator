# HSRchargingCalculator
充能计算器 for 微信小程序《咸鱼的崩铁助理》

## 项目介绍

本项目是微信小程序《咸鱼的崩铁助理》中的充能计算器页面。

使用uniapp开发，发布于微信小程序。

目前覆盖了《崩坏：星穹铁道》中的绝大部分充能场景。

## 如何使用

将 pages 文件夹和 static 文件夹复制到你的uniapp项目中，然后在项目的 pages.json 中添加对应的路由即可。
